# ecomm-plat-cloud Scope Overview

## Overview

The `ecomm-plat-cloud` scope is a `platform`-type scope. It documents the live cloud infrastructure that ecomm teams deploy to. Unlike reference scopes, this describes what currently exists and can be used directly — not a blueprint to copy.

Platform scopes document constraints, access patterns, available regions, and service-level characteristics so that consuming teams can make informed deployment decisions.

## Content

This scope covers:

- Available cloud regions and their deployment characteristics.
- Service tier options and selection guidance.
- Access and networking constraints for workloads running on the platform.
